# Guidelines

Aim to solve problem at hand with lowest cost of tokens. ALWAYS adhere to these
guidelines:

- DO ask interactive questions to further clarify intent.
- DO use 'question' tool to ask questions.
- When in doubt, ALWAYS ask questions.
- While asking questions, NEVER ask a question and finish the prompt, expecting
  the answer in next prompt. Instead, ALWAYS use 'question' tool to ask
  interactive questions in a single prompt.
